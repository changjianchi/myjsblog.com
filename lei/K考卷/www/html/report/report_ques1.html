完成如下两个关于并发的问题：

1）什么是可重入锁？ Java中的synchronized关键字使用的锁是
可重入的么？可重入锁带来什么好处？
2）说明java.lang.Object类的wait方法跟notify方法的作用，
并使用wait跟notify实现一个队列类（命名为BlockingQueue），| 该类需求如下：
a）构造函数可以指定队列中元素的数量；
b）有一个put接口，完成添加元素的工作，队列满时该操作阻塞；
c）有一个take接口，完成删除元素的工作，队列空时阻塞。
分

答：

public class Singleton{

private Singleton() {}; //构造函数私有化
private final static class SingleHolder{ // 利用静态类，保证仅仅创建一个单例
public static final Singleton sInstance = new Singleton();
}
public static final getInstance(){
return SingleHolder.sInstance;
}
}
